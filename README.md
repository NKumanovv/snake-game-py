# snake-game-py

Python Snake Game

Description:
A classic Snake game built using Python and Pygame. The player controls a snake that grows in length each time it eats the green food. The game ends if the snake hits the outer walls or its own tail.

How to use the project:
1. Ensure Python is installed on your computer.
2. Open a terminal in the project directory.
3. Install the required library by running:
   pip install -r requirements.txt
4. Launch the game by running:
   python main.py
5. Use the UP, DOWN, LEFT, and RIGHT arrow keys to control the snake.

Included Libraries:
- pygame: Used to create the graphical user interface, handle keyboard inputs, and draw shapes on the screen.
- random: Built-in library used to spawn the food at random locations.
